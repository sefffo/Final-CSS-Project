CSS Grid Layout 🗓️

See how grid-template-areas splits a page into named zones!

💡 What is grid-template-areas?

With grid-template-areas you can give each part of your page a name — like "header", "sidebar", "main" and "footer". Then CSS places every piece exactly where you drew it. It is like drawing a map of your page! 

.page-layout {
  display: grid;
  grid-template-areas:
    "header  header"
    "sidebar main  "
    "footer  footer";
}
